# Infrastructure as Code User Group Oslo

Welcome to the official GitHub organization of the **Infrastructure as Code User Group Oslo**!

## About Us
We are a community of enthusiasts, practitioners, and experts in Infrastructure as Code (IaC), based in Oslo, Norway. Our goal is to share knowledge, organize events, and foster collaboration around IaC best practices and tools.

## Mission
To connect and empower IaC professionals in Oslo and beyond.
